# Not production ready (roadmap coming soon)

## Panacea starter kit

Prerequisites:

* node + npm
* MongoDB

### Step 1
```
git clone git@github.com:panacea-js/starter-template
cd starter-template
```
### Step 2
```
npm run build:env
```
### Step 3
```
npm run build:cms
```
### Step 4
```
npm run start
```
### Step 5
Visit:
* Graphiql: [http://localhost:3000/graphiql]()
* Voyager: [http://localhost:3000/voyager]()
* Panacea CMS: [http://localhost:3000/cms]()

## Configuration

* Edit `panacea.js`
* Inspect `.env`
